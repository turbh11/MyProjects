# Assets Folder

הכנס כאן את הקבצים הבאים כדי לשפר את מראה הצעות המחיר:

## קבצים נדרשים:

### 1. רקע (אופציונלי):
- `background.jpg` - תמונת רקע לטקסט הצעת המחיר
- גודל מומלץ: 595x842 פיקסל (A4)
- שקיפות: 5% (המערכת מכניסה זאת אוטומטית)

### 2. לוגו (אופציונלי):
- `logo.png` - הלוגו של החברה
- גודל מומלץ: 80x60 פיקסל
- מיקום: פינה ימנית עליונה

### 3. חותמת (מומלץ):
- `stamp.png` - החותמת של המהנדס
- גודל מומלץ: 100x80 פיקסל
- מיקום: פינה ימנית תחתונה

### 4. חתימה (מומלץ):
- `signature.png` - החתימה האישית
- גודל מומלץ: 100x50 פיקסל
- מיקום: ליד החותמת

## הוראות:
1. שמור את הקבצים בשמות המדויקים המפורטים למעלה
2. הקבצים יתווספו אוטומטית להצעות המחיר החדשות
3. אם קובץ לא קיים, המערכת תמשיך לעבוד בלעדיו

## פורמטים נתמכים:
- תמונות: JPG, PNG, GIF
- רקע: JPG מומלץ
- חותמת וחתימה: PNG מומלץ (עם שקיפות)
